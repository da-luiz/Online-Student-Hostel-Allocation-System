# Online Student Hostel Allocation System (OSHAS)

**INS 204 – System Analysis and Design | Babcock University, School of Computing**
**Semester:** First Semester 2025/2026

## About

This repository contains the system analysis and design artefacts for OSHAS — a proposed web-based module that extends the existing University Management Information System (UMIS) to support room-level hostel allocation at Babcock University.

The current hostel allocation process requires students to be assigned rooms manually upon physical arrival at the hall. OSHAS addresses this by allowing students to browse available rooms, view room attributes (size, amenities, maintenance status), and reserve a room online before resumption.
